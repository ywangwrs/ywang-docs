# How to install Chinese input (shuangpin) in Ubuntu 20.04

## Reference
  - https://www.pinyinjoe.com/linux/ubuntu-18-gnome-chinese-setup.htm

## Installation
  - sudo apt-get install ibus-pinyin
  - sudo ibus restart
